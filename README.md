# AI Business Decision Engine

## 🚀 Problem
Businesses struggle to identify risky customers, churn probability, and profitability.

## 💡 Solution
This project builds an AI-based decision engine that predicts:
- Credit Risk
- Customer Churn
- Customer Segmentation

## 🛠️ Tech Stack
- Python
- Machine Learning
- Power BI
- Streamlit
